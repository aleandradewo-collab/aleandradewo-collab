## Featured projects

### 🏆 [World Cup 2026 Predictor](https://github.com/aleandradewo-collab/Prediccion_mundial_2026)
ML model that predicts the FIFA World Cup 2026 winner and top scorers using Poisson Regression, Gradient Boosting, Dixon-Coles ratings and 1,000 Monte Carlo simulations.
- 49K historical matches · 39 features · MAE ~0.93 goals
- Player-level predictions: top scorers, assisters and MVP probabilities
- Live web app with full bracket and team analysis

🌐 **[View live predictions →](https://aleandradewo-collab.github.io/Prediccion_mundial_2026/)**
